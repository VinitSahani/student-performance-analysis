Attendance vs. Academic Score Correlation

Raw Correlation (All Students) {{ analysis.correlation.corr_attendance_g3_all }} Includes G3 = 0 (absences)
Cleaned Correlation (Active Students) {{ analysis.correlation.corr_attendance_g3_active }} Excludes G3 = 0 (anomalous absences)
Analytical Interpretation:

Initially, calculating the correlation across all students yielded a coefficient of {{ analysis.correlation.corr_attendance_g3_all }}, suggesting zero relationship. This occurred because a significant group of students had zero absences but scored 0 in G3 (due to skipping the exam), which skewed the linear regression model.

By filtering out exam absences and focusing on active exam-taking students, the correlation rises to {{ analysis.correlation.corr_attendance_g3_active }} (a weak-to-moderate positive linear correlation). This proves that attending class has a direct statistical connection to higher final scores. Every class attended contributes positively to learning reinforcement and grade outcomes.
